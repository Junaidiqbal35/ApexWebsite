
# Apex Website

A project for Real Estate.



## Installation

Install project with venv

```bash
  python -m venv venv
  go to project folder
  pip Install -r requirements.txt
  python manage.py runserver
```
# Database
```
python manage.py makemigrations
python manage.py migrate
```
# Create Super User
